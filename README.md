# 🔮 FalaBak - React Native

**Teknoloji:** React Native — Expo SDK 54, Expo Router

**Styling:** React Native StyleSheet

**Dil:** TypeScript

---

# 📦 Kurulum

### Adımlar

Projeyi klonlayın:

```bash
git clone <repository-url>
```

Proje dizinine girin:

```bash
cd falabak
```

Bağımlılıkları yükleyin:

```bash
npm install
```

Uygulamayı çalıştırın:

```bash
npx expo start
```

Android:

```bash
npm run android
```

iOS (macOS gerekli):

```bash
npm run ios
```

# 🏗 Alınan Kararlar

* Uygulama verileri JSON dosyasından okundu.
* Tekrar kullanılabilir (Reusable) bileşen mimarisi oluşturuldu.
* Tema (renkler, tipografi ve spacing) merkezi bir dosyada yönetildi.
* Promotion geri sayımı JSON'daki `countdownDurationSeconds` değerine göre hesaplandı.
* Expo Router ile Tarot ekranına deep linking desteği sağlandı.
